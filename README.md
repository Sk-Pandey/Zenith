# ✦ Zenith

Zenith is a simple and modern task management application built with React. It helps users organize their daily tasks, track progress, and stay focused on what needs to be completed.

The project was created as a practice application to strengthen my understanding of React fundamentals, state management, component-based architecture, and modern UI design.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Search tasks in real time
- Visual progress tracking
- Clean and responsive user interface
- Modern dark-themed design

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- UUID

## Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/zenith.git
```

### Navigate to the project folder

```bash
cd zenith
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
